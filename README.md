# MTG Analyzer

A web-based Magic: The Gathering card analysis tool that helps players make informed decisions about their decks and card collections.

## 🌟 Features

- Interactive card search and display
- Detailed card analysis and statistics
- Modern, responsive web interface
- Real-time price tracking and market analysis

## 🚀 Live Demo

Visit the live application at: [MTG Analyzer](https://moonfallman.github.io/MTGAnalyzer/)

## 🛠️ Development

### Prerequisites

- Modern web browser
- Git for version control

### Local Development

1. Clone the repository:
```bash
git clone https://github.com/MoonfallMan/MTGAnalyzer.git
cd MTGAnalyzer
```

2. Branch Structure:
- `main` - Production branch, automatically deploys to GitHub Pages
- `dev` - Development branch for ongoing work

3. Development Workflow:
```bash
# Create a new feature branch from dev
git checkout dev
git checkout -b feature/your-feature-name

# Make your changes
# Commit your changes
git add .
git commit -m "Description of your changes"

# Push to remote repository
git push origin feature/your-feature-name

# Create a pull request to merge into dev
```
